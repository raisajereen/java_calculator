The following java code works as a calculator that performs arithmetic operations such as addition, substraction, multiplication, division, modulus (the remainder from a division). The calculator can also calculate the average of two inputted numbers, the maximum or minimum of two numbers and the square of a number. Once the code is finished running, all computed data is saved in a file with a name of the user's choice. The full code can be found in this depository as well as a test output file.
